# Vendor Performance Analysis Dashboard 📊

An interactive Power BI dashboard designed to analyze vendor performance across key business metrics including sales, profit, quantity, and customer behavior. Built using SQL and Power BI, this dashboard helps stakeholders make data-driven decisions with clear visual insights.

## Features
- KPI cards for total amount, quantity, profit, and average order value (AOV)
- Dynamic filters by quarter and category/vendor
- Bar charts showing:
  - Sales amount by state
  - Profit by month
  - Profit by sub-category
  - Customer-wise sales amount
- Pie charts for:
  - Quantity distribution by category
  - Payment mode breakdown by category

## Technologies Used
- SQL (for data extraction and transformation)
- Power BI (for data modeling and visualization)
- DAX (for calculated metrics and dynamic filtering)
